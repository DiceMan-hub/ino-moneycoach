新エリートランク「Diamond Reserve」の新設

💎 資格条件

年間宿泊数

80泊

年間利用額

$18,000

（約270-300万円）

※両方の条件を同時に満たす必要があります（AND条件）

導入背景：現在のDiamondステータスは、Amexカード保有者による獲得が容易になったため、会員数が急増し特典の価値が希薄化しています。Diamond Reserveは、真の高頻度・高額利用者を区別するための新ランクとなる可能性があります。

期待される特典：

  • 確約可能なスイートアップグレード（予約時に確約）
  • 保証されたレイトチェックアウト（午後4時など）
  • エグゼクティブラウンジへの優先アクセス
  • その他、競合プログラム（Marriott Ambassador、Hyatt Globalist）と同等の最上位特典

競合比較：Marriott Bonvoy Ambassador（100泊/$23,000）よりも条件が緩く、Hyatt Globalist（60泊）よりも宿泊数は多いが利用額条件は同程度です。

現状のGold・Diamond資格への影響：何が起こりうるか？

現在、リーク情報には矛盾があり、2つのシナリオが存在します。どちらが採用されるかは公式発表を待つ必要があります。

📉 シナリオA：資格条件が約30%緩和される場合

ステータス 現行条件（2025年） 緩和後（予想）
Silver 10泊 / 4滞在 / 25,000 BP 7泊 / 3滞在 / 17,500 BP
Gold 40泊 / 20滞在 / 75,000 BP 28泊 / 14滞在 / 52,500 BP
Diamond 60泊 / 30滞在 / 120,000 BP 42泊 / 21滞在 / 84,000 BP

起こりうる影響：

  • ステータス達成が容易になる可能性：より少ない宿泊数でGoldやDiamondを獲得できるようになる可能性があります
  • 会員数の急増：条件が緩和されると、Gold会員やDiamond会員の数が爆発的に増加する可能性があります
  • 特典の改悪リスク：会員数が増えると、ホテル側のコスト負担が増大する可能性があります。その結果、以下のような改悪が起こる可能性があります：
    • Gold会員の朝食無料特典の縮小または廃止
    • 飲食クレジット（F&Bクレジット）の金額削減
    • アップグレードの確率低下
    • エグゼクティブラウンジの混雑化
  • Amexプレミアムカードの価値低下：Diamondが42泊で達成可能になると、年間200万円のカード決済特典の価値が相対的に低下する可能性があります

📊 シナリオB：資格条件は変更なし（Diamond Reserveのみ追加）

現行の資格条件は維持される：

  • Silver: 10泊 / 4滞在 / 25,000 BP（変更なし）
  • Gold: 40泊 / 20滞在 / 75,000 BP（変更なし）
  • Diamond: 60泊 / 30滞在 / 120,000 BP（変更なし）

起こりうる影響：

  • 特典の優先順位の変化：Diamond Reserveが最上位になる可能性があるため、既存のDiamond会員は実質的に「準トップランク」に格下げされる可能性があります
  • アップグレード確率の低下：Diamond Reserve会員が最優先でアップグレードを受ける可能性があるため、既存Diamond会員のアップグレード確率が低下する可能性があります
  • ラウンジの混雑化：Diamond Reserve会員が優先的にラウンジを利用する可能性があるため、既存Diamond会員の利用機会が減少する可能性があります
  • 特典の改悪リスクは低い可能性：資格条件が変わらないため、会員数の急増は起こりにくく、現行の特典（Goldの朝食など）は維持される可能性が高いです

🔴 どちらのシナリオでも起こりうる影響

  • Diamond会員の相対的地位の低下：Diamond Reserveが最上位になる可能性があるため、既存のDiamond会員（特にAmexカードで獲得した会員）は、特典の優先順位において明確に「二番手」になる可能性があります
  • Amexプレミアムカードの価値への影響：年会費66,000円の根拠となっていた「最上位ステータス」が、もはや「最上位」ではなくなる可能性があります
  • 真の高額利用者の優遇：年間80泊かつ$18,000をホテルで支払う会員のみが、確約スイートや確約レイトチェックアウトなどの最優先特典を享受できるようになる可能性があります

📌 まとめ

現時点では、どちらのシナリオが採用されるかは不明です。しかし、どちらの場合でも、既存のGold・Diamond会員にとって重要な変化が起こる可能性があります。2025年中に予想される公式発表を注視することをおすすめします。

日本市場への影響：特にAmexプレミアムカード保有者へ

現状のDiamondステータス獲得方法

ヒルトン・オナーズ AMEX プレミアム・カード：年間200万円のカード決済でDiamondステータスを獲得（年会費66,000円）

※宿泊実績やホテルでの利用額とは無関係に、カード決済のみでステータス獲得が可能

Diamond Reserve導入後の影響

1. Diamond会員の二極化

  • （既存）Diamond会員：カード決済のみで獲得した会員、または60泊達成したが$18,000未満の会員
  • Diamond Reserve会員：80泊 かつ $18,000（約270-300万円）をホテルで支払った会員

2. 特典の優先順位の明確化

Diamond Reserve会員が、スイートアップグレード、ラウンジ利用、レイトチェックアウトなど、あらゆる特典において最優先で扱われる可能性があります。カード決済のみでDiamondになった会員は、実質的に「準トップランク」に格下げされる可能性があります。

3. Amexプレミアムカードの価値への影響

年会費66,000円の根拠となっていた「最上位ステータス」が、もはや「最上位」ではなくなる可能性があります。最優先の特典を期待する場合は、カード決済に加えて年間80泊かつ$18,000のホテル利用が必要になる可能性があります。

4. 一般カード保有のGold会員への影響

ヒルトン・オナーズ AMEX 一般カードなどでGoldステータスを獲得している会員についても、注意が必要です。プログラム変更により、Gold会員の朝食無料特典が縮小または廃止される可能性があります。特に、シナリオA（資格条件緩和）が実行される場合、Gold会員の特典改悪が伴うリスクがあります。一般カード保有者は、現在享受している朝食無料特典が利用できなくなる可能性があるため、今後の公式発表を注視することをおすすめします。

「The Honors Society」：招待制の最上位ランク

ランクの性質

招待制のみの最上位ランクとなる可能性があり、一般には公開されない可能性があります。ヒルトンが選定した会員のみが対象となる可能性があります。

✅ 一般の旅行客には直接的な影響はありません

想定される対象者

  • ホテルオーナー（ヒルトンブランドのフランチャイズ契約者）
  • 総支配人（GM）
  • 超VIP顧客（生涯宿泊数が数千泊に達するような極めて高額な利用実績を持つ顧客）
  • ヒルトンが独自の基準で選定した会員

戦略的な位置づけ

B2B関係性維持ツール：一般の旅行客（B2C）を対象としたロイヤルティプログラムの階層ではなく、ヒルトンのビジネス上、最も重要なパートナー（ホテルオーナーなど）を遇するために設計されたものとなる可能性があります。競合プログラム（Marriott Bonvoy Cobalt、World of Hyatt Courtesy Cardなど）にも同様の招待制ランクが存在します。

会員タイプ別の戦略的提言

Amexプレミアムカード保有者

現在のDiamondステータスは、近い将来「準トップランク」に格下げされる可能性があります。

提言：最優先の特典（確約スイート、確約レイトチェックアウト）を期待する場合は、カード決済に加えて年間80泊かつ$18,000（約270-300万円）のホテル利用を目指す必要があると予想されます。そこまでの利用をしない場合は、年会費66,000円の価値を再評価することをおすすめします。

高頻度・高額利用者（年間80泊/$18k超）

この変更が実現すれば、競合プログラム（Marriott Ambassador、Hyatt Globalist）と同等の真に差別化された特典を享受できる可能性が高まります。

Gold会員・宿泊ベースのDiamond会員（80泊/$18k未満）

既存ランクの資格条件緩和（シナリオA）が実行される場合、ステータス達成は容易になる可能性がありますが、特典の改悪（Goldの朝食特典の縮小など）を伴うリスクがあります。今後の公式発表を注視してください。
