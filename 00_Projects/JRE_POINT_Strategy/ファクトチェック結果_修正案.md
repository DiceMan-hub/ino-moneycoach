# JRE BANK併用メリット完全ガイド ファクトチェック結果と修正案

## 重大な修正が必要な箇所

### 1. JRE BANK優待割引券（4割引）の獲得枚数が不正確

**現在の記載（不正確）：**
- 50万円以上 + ビューカード引落のみ → 2枚
- 50万円以上 + 給与受取のみ → 2枚
- 50万円以上 + 両方 → 2枚
- 300万円以上 + ビューカード引落のみ → 4枚
- 300万円以上 + 給与受取のみ → 4枚
- 300万円以上 + 両方 → 4枚

**正しい情報：**
- 50万円以上 + ビューカード引落のみ → **1枚**
- 50万円以上 + 給与受取のみ → **2枚**
- 50万円以上 + 両方 → **3枚**
- 300万円以上 + ビューカード引落のみ → **2枚**
- 300万円以上 + 給与受取のみ → **3枚**
- 300万円以上 + 両方 → **5枚**
- 判定日：6月25日と12月25日の年2回
- 年間最大：**10枚**

**修正箇所：**
- 第4章 4.2節のマトリクステーブル
- 第5章 5.1節の「ベースラインで得られる特典」の説明
- 第5章 5.2節のROIシミュレーション（年間2枚ではなく1枚に修正）

**影響：**
- ROI計算が過大評価されている（年間価値が約14,136円→約7,068円に減少）
- 総合的な年間価値が約25,336円→約18,268円に減少
- 実質利回りが約5.06%→約3.65%に減少

### 2. 特典の受取方法が変更されている

**現在の記載：**
- 「えきねっと」のマイページにて、プレゼントされた優待割引券番号を登録する手続きが必要

**正しい情報（2025年6月9日以降）：**
- JRE BANKアプリ・ウェブサイト内の「特典受取」画面から受け取る
- メール通知からアプリ内受取に変更

**修正箇所：**
- 第3章 3.1節「利用方法」の説明

**修正案：**
```
<strong>受取方法：</strong>JRE BANKアプリまたはウェブサイトにログインし、「特典受取」画面から受け取る。2025年6月9日以降、メール通知からアプリ内受取に変更されている。
<strong>利用方法：</strong>受け取った優待割引券は「えきねっと」からのみ申込み可能。事前に「えきねっと」のマイページにて、優待割引券番号を登録する手続きが必要となる。
```
