# レポート調からブログ調への変換プロンプト

## 変換の基本方針

### レポート調の特徴（変換前）
- 「本分析では」「概観します」「定義します」などの堅苦しい表現
- 「試算できます」「分析結論」などの学術的な表現
- 読者との距離感がある
- 客観的で距離のある語り口

### ブログ調の特徴（変換後）
- 読者に語りかける親しみやすい表現
- 「見ていきましょう」「考えてみましょう」などの自然な表現
- 読者との距離感を縮める
- 親近感のある語り口

## 具体的な変換ルール

### 1. 導入部分の変換
- 「本分析では」→「このガイドでは」「今回は」
- 「概観します」→「見ていきましょう」「ご紹介します」
- 「定義します」→「考えてみましょう」「お伝えします」

### 2. 説明部分の変換
- 「試算できます」→「計算してみると」「試算してみましょう」
- 「分析します」→「詳しく見ていきましょう」「解説します」
- 「決定されます」→「決まります」「設定されています」

### 3. 結論部分の変換
- 「分析結論」→「まとめると」「結論として」
- 「導き出される」→「分かります」「言えます」
- 「みなされます」→「考えられます」「言えます」

### 4. 読者への語りかけ
- 「利用者にとって」→「あなたにとって」「みなさんにとって」
- 「必要があります」→「必要です」「必要になります」
- 「可能です」→「できます」「可能になります」

### 5. 自然な表現への変換
- 「多岐にわたります」→「たくさんあります」「様々です」
- 「潜在能力を持ちます」→「可能性があります」「期待できます」
- 「劇的に上昇します」→「大きく上がります」「大幅にアップします」

### 6. 親しみやすい表現
- 「非常に効率的な」→「とてもお得な」「かなり効率的な」
- 「圧倒的な」→「すごい」「非常に高い」
- 「紛れもなく」→「間違いなく」「確実に」

## 変換のポイント

1. **読者との距離を縮める**
   - 一人称・二人称を適切に使用
   - 「あなた」という呼びかけを自然に使う

2. **専門用語を分かりやすく**
   - 専門用語の後に簡単な説明を追加
   - 例：「フロア価値（最低保証価値）」

3. **文章のリズムを整える**
   - 長い文章を短く分割
   - 読点を適切に使用

4. **感情を込める**
   - 「すごい」「お得」「便利」などの感情的な表現を適度に使用
   - ただし、過度にならないよう注意

5. **具体例を増やす**
   - 「例えば」を多用
   - 読者がイメージしやすい具体例を追加

## 適用範囲

- 本文のすべての段落
- 見出しは変更しない（構造を維持）
- 数値や計算式はそのまま維持
- 表やリストの内容も自然な表現に変換
