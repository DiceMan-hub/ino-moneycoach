# アメックス フリーステイギフト × ザ・ホテル・コレクション プロジェクト

## 概要
アメリカン・エキスプレス プラチナカードの「フリーステイギフト」と「ザ・ホテル・コレクション」の両方の特典が利用可能な厳選ホテルを紹介するWebページです。

## ファイル構成

1. **ウェスティンホテル横浜** - 横浜
2. **ハイアット リージェンシー 瀬良垣アイランド 沖縄** - 沖縄
3. **ヒルトン京都** - 京都
4. **ヒルトン沖縄瀬底リゾート** - 沖縄

- プラチナ（個人）
